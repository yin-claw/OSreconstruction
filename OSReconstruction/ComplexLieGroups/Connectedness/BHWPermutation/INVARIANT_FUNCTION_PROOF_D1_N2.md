# Proof: BHW Permutation Symmetry for d=1, n=2 via Lorentz Invariant Functions

Last updated: 2026-03-02

## Claim

For d=1, n=2, swap τ = (0,1):

Given F satisfying holomorphy on FT, real restricted Lorentz invariance, boundary
values, and local commutativity (the standard blocker hypotheses), we have:

> For all z ∈ FT with swap·z ∈ ET: extendF(swap·z) = F(z)

This closes `blocker_eventuallyExtendFBaseEq_on_prepared_nhds_at_anchor_adjSwap_d1_swap01_n2`
(and in fact gives a stronger global result, not just local).

## Key Observation

`complex_lorentz_invariance` is **fully proved** (zero sorrys) in
`ComplexInvariance/Extend.lean`. It gives:

> For all Λ ∈ SO(2,ℂ), z ∈ FT, Λ·z ∈ FT: F(Λ·z) = F(z)

This was proved using connectedness of SO(2,ℂ) (which is path-connected), NOT
using orbit-FT connectivity. The proof shows the "invariance set"
T = {Λ : ∀ w ∈ FT, Λw ∈ FT → F(Λw) = F(w)} is closed, contains 1, and its
complement (intersected with the non-vacuous part) is open, so T = SO(2,ℂ) by
connectedness.

**Crucially**: this gives F(Λz) = F(z) for EVERY Λ with z, Λz ∈ FT. So F is
constant on orbits within FT, even if those orbits are disconnected within FT.

## Proof

### Step 1: Lorentz Invariant Ring for SO(2,ℂ) on (ℂ²)²

Write z = (z₀, z₁) with z_k = (z_{k0}, z_{k1}) ∈ ℂ². Use light-cone coordinates:

- u_k = z_{k0} + z_{k1}, v_k = z_{k0} - z_{k1}

The complex Lorentz group SO(2,ℂ) acts as: λ · (u_k, v_k) = (λ u_k, λ⁻¹ v_k)
for λ ∈ ℂ* (the isomorphism SO(2,ℂ) ≅ ℂ*).

The invariant ring for ℂ* acting diagonally on (u₀, v₀, u₁, v₁) by
(u₀, v₀, u₁, v₁) ↦ (λu₀, λ⁻¹v₀, λu₁, λ⁻¹v₁) is generated by:

- Q₀ = u₀v₀ = z₀₀² - z₀₁² (Minkowski norm of particle 0)
- Q₁ = u₁v₁ = z₁₀² - z₁₁² (Minkowski norm of particle 1)
- R = u₀v₁ = (z₀₀ + z₀₁)(z₁₀ - z₁₁)
- T = u₁v₀ = (z₁₀ + z₁₁)(z₀₀ - z₀₁)

subject to the single relation: R · T = Q₀ · Q₁.

Alternative invariant set: (Q₀, Q₁, P, S) where:
- P = ½(R + T) = z₀₀z₁₀ - z₀₁z₁₁ (Minkowski inner product)
- S = R - T = 2(z₀₁z₁₀ - z₀₀z₁₁) = -2 det([z₀ | z₁])

The relation becomes: S² = (R - T)² = (R + T)² - 4RT = 4P² - 4Q₀Q₁.

So (Q₀, Q₁, P, S) live on the variety V = {S² = 4(P² - Q₀Q₁)} ⊂ ℂ⁴.

### Step 2: Factorization of F through Invariants

By `complex_lorentz_invariance`, F is constant on SO(2,ℂ)-orbits within FT.

**Key fact on FT**: For z ∈ FT (d=1), Im(z₀) ∈ V⁺ implies z₀ is non-lightlike,
i.e., Q₀ = u₀v₀ ≠ 0. This is because Im(z₀₀) > |Im(z₀₁)| implies
Im(u₀) = Im(z₀₀) + Im(z₀₁) > 0 and Im(v₀) = Im(z₀₀) - Im(z₀₁) > 0,
so u₀ ≠ 0 and v₀ ≠ 0, hence Q₀ ≠ 0.

Since Q₀ ≠ 0 on FT, the stabilizer of z₀ under ℂ* is trivial (λu₀ = u₀ and
λ⁻¹v₀ = v₀ with u₀, v₀ ≠ 0 forces λ = 1). Therefore SO(2,ℂ)-orbits on FT are
free, and the invariant map π : FT → V is a submersion at generic points.

Since F is constant on π-fibers (orbits) and π is a submersion on an open dense
set, there exists a holomorphic function f on the image π(FT) ⊂ V such that:

> F(z) = f(Q₀(z), Q₁(z), P(z), S(z)) on the generic locus

By holomorphic extension (the generic locus is open and dense in the connected
set FT), this holds on all of FT:

> **F = f ∘ π on FT**

### Step 3: Swap Action on Invariants

Under swap (z₀, z₁) ↦ (z₁, z₀):

- Q₀ ↦ Q₁, Q₁ ↦ Q₀ (norms swap)
- P ↦ P (inner product is symmetric)
- S = -2 det([z₀|z₁]) ↦ -2 det([z₁|z₀]) = 2 det([z₀|z₁]) = -S

So: **swap acts on invariants as σ : (q₀, q₁, p, s) ↦ (q₁, q₀, p, -s)**.

Note this preserves the variety V: s² = 4(p² - q₀q₁) ⟺ (-s)² = 4(p² - q₁q₀).

### Step 4: Lorentz Action on Invariants

For Γ ∈ SO(2,ℂ) = ComplexLorentzGroup(1):

- Q_k(Γz_k) = Q_k(z_k) (Lorentz preserves norms)
- P(Γz₀, Γz₁) = P(z₀, z₁) (Lorentz preserves inner product)
- S(Γz₀, Γz₁) = -2 det(Γ[z₀|z₁]) = -2 det(Γ) det([z₀|z₁]) = S(z₀, z₁)
  (since det(Γ) = 1 for Γ ∈ SO(2,ℂ))

So Lorentz transformations preserve ALL invariants, as expected.

### Step 5: Local Commutativity in Invariant Coordinates

`hF_local` says: at real points x = (x₀, x₁) with (x₁ - x₀) spacelike
(i.e., Q₀ + Q₁ - 2P < 0), F(x₀, x₁) = F(x₁, x₀).

In invariant coordinates: f(Q₀, Q₁, P, S) = f(Q₁, Q₀, P, -S) on the real
image of spacelike configurations.

Define: **g(q₀, q₁, p, s) = f(q₀, q₁, p, s) - f(q₁, q₀, p, -s)**

g is holomorphic on V (smooth part), and g = 0 on the real spacelike image.

### Step 6: The Real Spacelike Image Contains a Totally Real Open Set

**Claim**: The image of real spacelike configurations (x₀, x₁) ∈ ℝ² × ℝ²
under π = (Q₀, Q₁, P, S) contains a 3-dimensional totally real submanifold
of V.

**Proof**: Consider the map (a, b, c, d) ↦ (Q₀, Q₁, P, S) where
z₀ = (a, b), z₁ = (c, d) are real. The Jacobian of (Q₀, Q₁, P, S) w.r.t.
(a, b, c, d) is:

```
 ∂Q₀/∂a = 2a    ∂Q₀/∂b = -2b   ∂Q₀/∂c = 0     ∂Q₀/∂d = 0
 ∂Q₁/∂a = 0     ∂Q₁/∂b = 0     ∂Q₁/∂c = 2c    ∂Q₁/∂d = -2d
 ∂P /∂a = c     ∂P /∂b = -d    ∂P /∂c = a     ∂P /∂d = -b
 ∂S /∂a = -2d   ∂S /∂b = 2c    ∂S /∂c = 2b    ∂S /∂d = -2a
```

At the spacelike point x₀ = (1, 0), x₁ = (1, 2) (spacelike since
(x₁-x₀)² = 0² - 2² = -4 < 0):

Invariants: Q₀ = 1, Q₁ = 1-4 = -3, P = 1, S = 2(0·1 - 1·2) = -4.
Check: S² = 16 vs 4(P² - Q₀Q₁) = 4(1 + 3) = 16. ✓

Jacobian at this point:
```
 [2, 0, 0, 0]
 [0, 0, 2, -4]
 [1, -2, 1, 0]
 [-4, 2, 0, -2]
```

This 4×4 matrix maps ℝ⁴ to ℝ⁴. The image lies on V (by construction),
which is a 3-real-dimensional surface within ℝ⁴. The rank of this matrix
restricted to directions tangent to V determines the dimension of the
real spacelike image.

V is defined by h(q₀,q₁,p,s) = s² - 4p² + 4q₀q₁ = 0. At our point:
∇h = (4q₁, 4q₀, -8p, 2s) = (-12, 4, -8, -8).
So the tangent space to V is the 3-dimensional kernel of ∇h.

The 4×4 Jacobian above has rank at least 3 (the 3×3 minor from rows 1,2,4
and columns 1,3,4 is det = 2·(2·(-2)-(-4)·0) - 0 + 0 = 2·(-4) = -8 ≠ 0).

Since the Jacobian has rank ≥ 3 and the target tangent space T_V is
3-dimensional, the image of ℝ⁴ in T_V is 3-dimensional. So the real
spacelike image contains a 3-dimensional real manifold inside V. ✓

This is a **maximally totally real** submanifold of V (a 3-complex-dimensional
variety).

### Step 7: Identity Theorem on V

g is a holomorphic function on the smooth part of V (a connected 3-complex-
dimensional manifold) that vanishes on a 3-real-dimensional totally real
submanifold. By the **identity theorem for holomorphic functions on complex
manifolds** (a holomorphic function vanishing on a maximally totally real
submanifold of a connected complex manifold vanishes identically):

> **g ≡ 0 on V**

That is: **f(q₀, q₁, p, s) = f(q₁, q₀, p, -s) for all (q₀, q₁, p, s) ∈ V**.

### Step 8: Closing the Blocker

For z ∈ FT with swap·z ∈ ET, let Γ ∈ SO(2,ℂ) satisfy Γ·(swap·z) ∈ FT.
By well-definedness of extendF:

extendF(swap·z) = F(Γ·(swap·z))

Now Γ·(swap·z) = (Γz₁, Γz₀) ∈ FT. Its invariants (by Steps 3-4):

- Q₀(Γz₁, Γz₀) = Q₁(z₀, z₁)  [Lorentz preserves norms, swap exchanges]
- Q₁(Γz₁, Γz₀) = Q₀(z₀, z₁)
- P(Γz₁, Γz₀) = P(z₀, z₁)
- S(Γz₁, Γz₀) = -S(z₀, z₁)    [Lorentz preserves S, swap negates it]

Therefore:

```
extendF(swap·z) = F(Γz₁, Γz₀)              [well-definedness of extendF]
               = f(Q₁, Q₀, P, -S)           [F = f ∘ π, Step 2]
               = f(Q₀, Q₁, P, S)            [Step 7: identity on V]
               = F(z₀, z₁)                  [F = f ∘ π, Step 2]
               = F(z)
```

**QED.** ∎

## Remarks

### This proves MORE than the blocker asks for

The blocker asks for local eventual equality on a prepared neighborhood. We get
**global** equality: extendF(swap·z) = F(z) for ALL z ∈ FT with swap·z ∈ ET.
So the prepared-neighborhood / anchor / Γ hypotheses are not needed at all.

### Why the previous routes failed

All previous proof attempts tried to establish the equality WITHOUT going through
complex Lorentz invariance (which was thought to have its own blockers). But
`complex_lorentz_invariance` is already fully proved! The invariant-function
route uses it as a black box to reduce to a 3-dimensional identity theorem
problem, completely bypassing:
- Real Jost point witnesses (don't exist in d=1)
- Back-witnesses Λ with Λ·(swap·z) = z (nongeneric in d=1)
- Common forwardizers (formally impossible in d=1, n=2)
- FT ∩ swap·FT overlap (empty in d=1)
- Midpoint implication chains (formally counterexampled in d=1)
- Orbit-FT connectivity (a separate blocker, NOT needed here)

The key insight is that `complex_lorentz_invariance` gives F constant on orbits
within FT **regardless of orbit connectivity**, because it gives the identity
F(Λz) = F(z) for **every** Λ ∈ SO(2,ℂ), not just for Λ connected to 1 within
the orbit-FT intersection.

### Generalization to n > 2

For n > 2, the same approach works:
1. Complex Lorentz invariance → F factors through SO(2,ℂ)-invariants on FT
2. Invariants: Q_k = z_{k,0}² - z_{k,1}² and cross-terms P_{jk}, S_{jk}
3. Adjacent swap (i, i+1) acts by: swap Q_i ↔ Q_{i+1}, etc., and negate S_{i,i+1}
4. Local commutativity → identity on real spacelike image → identity everywhere
5. Each adjacent swap gives extendF(swap_i · z) = F(z)
6. General permutation by induction on adjacent transpositions

For n > 2, the induction composition (Step 6) still needs the existing
PermutationFlow.lean induction machinery for composing adjacent swaps on the
extended tube. But the **base case** (each individual adjacent swap) is handled
by this invariant-function argument.
